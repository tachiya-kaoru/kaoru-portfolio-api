# 学習記録管理アプリ（Learning Record App）

> 自分の学習時間を「記録」し、「継続力」と「熱意」を可視化するための学習記録管理アプリです。
> プログラミング学習だけでなく、**エラーと格闘した時間・セミナー受講時間・書籍を読んだ時間**も学習として積み上げられることを目指しています。

## 現在の主な機能

| 機能 | 内容 |
| --- | --- |
| ユーザー登録 / ログイン | セッションベースの認証、パスワードは BCrypt でハッシュ化 |
| プロフィール編集 | 自己紹介の編集、アバター画像のアップロード（AWS S3 保存） |
| 学習記録の登録 | 月・カテゴリごとに学習項目と学習時間を登録 |
| バリデーション | 入力チェック（必須・文字数・数値）と重複登録の防止 |
| 学習記録の一覧表示 | 月・カテゴリ別に登録済みの学習項目を表示 |

## 設計上の工夫

### 画像保存を S3 に切り出して「疎結合」に

現在はバックエンドの開発に重点を置いているため、デプロイ先には Render を利用しています。
ただし、**将来的に本番サービスとして運用することを見据え、インフラの学習を兼ねてAWS への移行も視野に入れています。**

その第一歩として、**アバター画像の保存をアプリ本体から切り離し、AWS S3 に保存する構成**にしました。
画像の保存先をアプリのファイルシステムに依存させないことで、

- デプロイ環境を移しても画像が消えない
- 後から AWS へ移行しやすい

といった「疎結合」なつくりを意識しています。

## 制作中の想い
課題として、学習時間を可視化しプロフィールに載せる。というアプリを制作する中で、もっと実際の学習時間を反映できるものにしたいと考えるようになりました。
未経験からの転職活動では、資格・ポートフォリオ・GitHub も実績になるとは考えていますが、
それらと合わせて
- エラーと格闘した時間
- セミナーを受講した時間
- 技術書を読んだ時間
など、成果物やコミット歴だけでは伝わらない時間も、プログラミング学習であり自分の財産であると考えているからです。
自分自身の学習・転職だけでなく、今後同じように未経験でエンジニアを目指す方の**継続力と熱意を伝えるための一つのツール**として、このアプリを育てていきたいと考えています。

## 独自の追加・開発予定

### 1.公開用ページの追加
現在はログインした本人のみプロフィールを参照できる仕様ですが、登録したプロフィールを共有するための
公開用ページを実装したいと考えています。

### 2.ポモドーロタイマーとの連携

現在は学習記録を手入力していますが、
普段の学習で使っているポモドーロタイマーと連携し、**計測した学習時間をそのまま記録できる**ようにしたいと考えています。

### 3. AWS への移行

本番運用を見据え、デプロイ環境を Render から AWS へ移行することを検討しています。

## 環境構築手順（ローカル）

> ※ 開発環境のセットアップ手順です。

### 1. リポジトリのクローン

```bash
git clone https://github.com/tachiya-kaoru/kaoru-portfolio-api
```

### 2. コンテナの起動

```bash
docker compose up -d
```

### 3. アプリの起動

```bash
./gradlew bootRun
```

### 4. アクセス

ブラウザで以下にアクセスします。

```
http://localhost:8080/
```
